# 🛡️ superMen — RPG для прокачки реального себя

> Прокачивай реального себя — и твой онлайн-персонаж растёт вместе с тобой.

**superMen** — это геймификация саморазвития в стиле классической RPG. Ты ходишь
на английский, испанский, в спортзал, работаешь, ищешь приключения — а приложение
превращает эти реальные действия в опыт (XP), очки характеристик, золото, квесты и
ачивки. Персонаж качается, экипируется, получает уровни и ранги. Дофаминовые
триггеры (анимации левел-апа, дроп лута, пламя стриков, ежедневные сводки) удерживают
мотивацию делать важное в реальной жизни.

## 🧩 Стек

| Слой           | Технология                                              |
|----------------|---------------------------------------------------------|
| Клиент         | **React + TypeScript**, упакован как **Telegram Mini App** |
| Бэкенд         | **Go** (REST API + cron-шедулер нотификаций)            |
| База данных    | **PostgreSQL**                                          |
| Идентификация  | Telegram `initData` (без экрана логина)                |
| Нотификации    | **Telegram Bot API** (ежедневная сводка)               |

> На первом этапе **отдельной авторизации нет**: пользователь идентифицируется
> автоматически через Telegram, при открытии Mini App сразу попадает на главный экран.

## 🚦 Статус

Реализован **рабочий вертикальный срез (Фаза 0–1+)**: Go-бэкенд на стандартной
библиотеке (REST API + initData-auth + Telegram-бот + cron-шедулер), React TMA
(экраны Герой/Квесты/Магазин/Профиль/Lab, чек-ин с анимациями наград, CharacterCanvas),
60 плейсхолдер пиксель-спрайтов + генератор, миграции PostgreSQL и сиды.

Бэкенд **запускается офлайн с in-memory стором** (без БД) — для дог-фудинга; Postgres
подключается через `DATABASE_URL`. См. [роадмап](./docs/11-roadmap.md) и [запуск](./docs/13-running.md).

## 🏁 Quickstart

```bash
# backend — REST API на :8080, in-memory стор (без БД), dev-режим
make dev
#   эквивалент: ENV=dev DATABASE_URL= go run .

# frontend — React TMA (Vite dev-сервер)
make web-dev
#   эквивалент: cd web && npm install && npm run dev

# пиксель-арт спрайты (плейсхолдеры) → web/public/assets/character/
make genassets

# с PostgreSQL: поднять БД + применить миграции
make up && make migrate     # docker compose + migrations/*.sql
```

Быстрая проверка API (dev-режим использует `X-Device-Id` вместо Telegram initData):

```bash
curl -H "X-Device-Id: me-1" localhost:8080/api/v1/me
curl -H "X-Device-Id: me-1" -H "Content-Type: application/json" \
     -d '{"activityKey":"english","durationMin":30}' localhost:8080/api/v1/checkin
```
