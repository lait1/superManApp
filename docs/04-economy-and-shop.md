# 04 — Экономика и магазин

← [03 — Прогрессия и статы](./03-progression-and-stats.md) · далее → [05 — UI/UX](./05-ui-ux.md)

Золото — внутриигровая валюта. **Тратится только на виртуальную экипировку** (косметика +
шмот, влияющий на статы/внешний вид персонажа). Реальных денег и реальных покупок нет.

---

## 1. Источники золота (sources)

| Источник | Сколько |
|----------|---------|
| Чек-ин активности | базовое золото активности (см. [02](./02-activities-and-quests.md)) |
| Бонус за стрик | до +50% к золоту (`GOLD_STREAK_BONUS`, см. [03](./03-progression-and-stats.md)) |
| Завершение квеста | фиксированная награда квеста |
| Получение ачивки | разовый бонус |
| Level-up / ранг-ап | бонус золота за веху |

## 2. Стоки золота (sinks)

| Сток | На что |
|------|--------|
| Покупка экипировки | оружие, броня, амулеты, фоны, ауры |
| Косметика | скины, рамки, титульные эффекты |
| **Streak Freeze** | защита стрика от обрыва (расходник) |
| Перекраска / реролл стиля *(v1+)* | косметические изменения |

> Баланс «source vs sink» важен против инфляции: если золото копится быстрее, чем тратится,
> цели обесцениваются. Регуляторы — цены в магазине и `base_gold` активностей.

## 3. Экипировка: слоты

У персонажа есть слоты экипировки. Надетые предметы меняют **внешний вид** и могут давать
**пассивные эффекты**. Как именно слоты накладываются на персонажа визуально (paper-doll,
порядок слоёв, редкость) — см. [12 — Дизайн персонажа](./12-character-design.md).

```
        ┌───────────────┐
        │   🎯 Аура       │  (эффект/класс)
        ├───────┬───────┤
  ⚔️ Оружие │  ПЕРС  │ 🛡️ Броня
        ├───────┴───────┤
        │  📿 Амулет      │  (бусты)
        ├───────────────┤
        │  🖼️ Фон (сцена) │  (косметика)
        └───────────────┘
```

| Слот | Тип эффекта |
|------|-------------|
| ⚔️ Оружие | косметика + малый буст XP в домене |
| 🛡️ Броня | косметика + защита стрика / VIT-бонус |
| 📿 Амулет | буст золота или XP |
| 🎯 Аура | визуальный эффект класса/ранга |
| 🖼️ Фон | чистая косметика (сцена за персонажем) |

## 4. Редкость

| Редкость | Цвет | Эффекты | Шанс дропа |
|----------|------|---------|-----------|
| Common (Обычный) | ⚪ серый | косметика | высокий |
| Uncommon (Необычный) | 🟢 зелёный | слабый буст | средний |
| Rare (Редкий) | 🔵 синий | заметный буст | низкий |
| Epic (Эпический) | 🟣 фиолетовый | сильный буст | редкий |
| Legendary (Легендарный) | 🟠 оранжевый | сильный буст + уник. визуал | очень редкий / квестовый |

Легендарки часто **не покупаются**, а выдаются за эпические квесты/ачивки — чтобы они
были маркером реальных достижений, а не кошелька.

## 5. Эффекты экипировки (примеры)

```yaml
- id: amulet_owl
  name: "Амулет Совы"
  slot: amulet
  rarity: rare
  effect: { type: xp_mult, stat: INT, value: 0.10 }   # +10% XP к Интеллекту
  price: 1200

- id: armor_titan
  name: "Доспех Титана"
  slot: armor
  rarity: legendary
  effect: { type: streak_shield, charges: 1 }           # авто-защита стрика раз в неделю
  source: quest_gym_50                                   # только за квест, не в продаже

- id: bg_neon_city
  name: "Неоновый город"
  slot: background
  rarity: uncommon
  effect: { type: cosmetic }
  price: 400
```

**Порядок применения бустов** (из [03](./03-progression-and-stats.md)):
`base → стрик → крит → длительность → сумма бустов экипировки`.

Бусты складываются аддитивно и ограничены потолком (напр. суммарный XP-буст ≤ +40%),
чтобы экипировка усиливала, но не ломала экономику.

## 6. Streak Freeze (расходник)

- Покупается за золото (напр. 300).
- Хранится в инвентаре, максимум 2 шт.
- Автоматически тратится, если день пропущен, спасая стрик.
- Логика стрика — в [01 — Механики §5](./01-game-mechanics.md#5-стрики-серии-дней-).

## 7. Магазин: разделы

```
┌──────────────────────────────────────────┐
│  🏪 МАГАЗИН            💰 6 420            │
│  [ Снаряжение ][ Косметика ][ Расходники ] │
│                                            │
│  🔵 Амулет Совы           1 200  💰 [Купить]│
│  🟣 Клинок Фокуса         3 500  💰 [Купить]│
│  🟢 Неоновый город          400  💰 [Купить]│
│  🧊 Заморозка стрика        300  💰 [Купить]│
└──────────────────────────────────────────┘
```

Купленный предмет уходит в **инвентарь**; оттуда его можно **надеть** в слот.
Эндпоинты покупки/экипировки — в [09 — API](./09-api.md).

## 8. Защита от «pay-to-win» (даже без реальных денег)

Поскольку золото зарабатывается только реальными действиями, «богатство» = «усердие».
Но чтобы шмот не превращался в самоцель в ущерб реальному прогрессу:
- бусты ограничены потолком;
- лучшие предметы — за достижения, не за накопление;
- косметика (большая часть магазина) не даёт механических преимуществ.
