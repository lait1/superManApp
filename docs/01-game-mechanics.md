# 01 — Игровые механики

← [00 — Видение](./00-vision.md) · далее → [02 — Активности и квесты](./02-activities-and-quests.md)

Это ядро геймификации. Здесь описаны характеристики, опыт и уровни, классы,
стрики и ранги. Конкретные формулы вынесены в [03 — Прогрессия и статы](./03-progression-and-stats.md).

---

## 1. Характеристики (статы)

Пять RPG-атрибутов, каждый привязан к домену реальной жизни. Это позволяет одним
взглядом понять, в чём ты прокачан, а где «провисаешь».

| Стат | Иконка | Домен реальной жизни | Качается активностями |
|------|--------|----------------------|------------------------|
| **Сила** (STR) | 💪 | Тело, спорт | спортзал, бег, плавание, спорт |
| **Интеллект** (INT) | 🧠 | Знания, языки | английский, испанский, чтение, курсы |
| **Дисциплина** (DIS) | 🎯 | Работа, рутина | работа, фокус-сессии, ранний подъём |
| **Жизненная сила** (VIT) | ❤️ | Здоровье, восстановление | сон, питание, медитация, вода |
| **Харизма** (CHA) | ✨ | Социальное, новый опыт | приключения, встречи, выступления |

Каждый стат имеет:
- **Значение** (целое число) — растёт от активностей.
- **Уровень стата** — производное от значения (см. кривую в [03](./03-progression-and-stats.md)).
- **Прогресс-бар** до следующего уровня стата.

> Статы — не только цифры. Они влияют на **внешний вид персонажа** (поза, аура,
> доступная экипировка) и открывают **квесты и ачивки**, завязанные на конкретный домен.

## 2. Опыт (XP) и уровни

Есть два уровня прогрессии:

### Уровень персонажа (общий)
Складывается из суммарного XP по всем активностям. Это «главная цифра» героя.

```
┌─────────────────────────────────────────────┐
│  ⚔️  superMen          LVL 14                  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░░░  6 820 / 9 100 XP       │
└─────────────────────────────────────────────┘
```

### Уровни статов (по доменам)
Каждый стат растёт отдельно. Можно быть «LVL 20 Интеллект, LVL 6 Сила» — это честно
показывает дисбаланс и подталкивает докачать отстающее (квест «Сбалансированный герой»).

Формула кривой (детали в [03](./03-progression-and-stats.md)):

```
xp_to_next(level) = BASE * level^1.5      // BASE ≈ 100
```

→ ранние уровни берутся быстро (быстрый дофамин на старте), дальше — медленнее (long-term).

## 3. Крит и переменная награда

Чтобы чек-ин не был предсказуемым:
- **Крит XP** (шанс ~10%): ×2 опыта за активность, с отдельной анимацией.
- **Дроп шмота** (шанс зависит от редкости активности): после чек-ина может «выпасть»
  предмет экипировки. См. [04 — Экономика](./04-economy-and-shop.md).

Это и есть «переменное вознаграждение» из [видения](./00-vision.md) — главный крючок удержания.

## 4. Классы / архетипы

Класс **не выбирается вручную** — он вычисляется по доминирующему стату и эволюционирует
вместе с тобой. Это работает на идентичность («я — Мудрец»).

| Доминирующий стат | Класс | Флейвор |
|-------------------|-------|---------|
| 💪 Сила | **Воин** (Warrior) | сила, выносливость |
| 🧠 Интеллект | **Мудрец** (Sage) | знания, языки |
| 🎯 Дисциплина | **Паладин** (Paladin) | порядок, воля |
| ❤️ Жизненная сила | **Друид** (Druid) | баланс, здоровье |
| ✨ Харизма | **Бард** (Bard) | люди, приключения |
| Всё равномерно (±) | **Авантюрист** (Adventurer) | универсал |

Класс даёт: косметику (визуальный стиль), небольшие классовые квесты и титулы.
Гибридные классы (две сильные характеристики) — на v2.

## 5. Стрики (серии дней) 🔥

Стрик = число подряд идущих дней, в которые был хотя бы один чек-ин.

- **Множитель XP за стрик** растёт по дням и упирается в потолок:

  | Стрик | Множитель XP |
  |-------|--------------|
  | 1–2 дня | ×1.0 |
  | 3–6 дней | ×1.1 |
  | 7–13 дней | ×1.25 |
  | 14–29 дней | ×1.4 |
  | 30+ дней | ×1.5 (максимум) |

- **Заморозка стрика (Streak Freeze)** — предмет за золото. Защищает стрик при
  пропуске одного дня. Можно держать до 2 штук. Это снимает тревожность и не даёт
  «бросить из-за одной осечки».
- Пропуск без заморозки → стрик сбрасывается в 0 (но **накопленный XP, золото, статы и шмот
  не теряются** — наказание мягкое, см. принцип №5 в [видении](./00-vision.md)).

Стрики бывают **общие** (любая активность) и **по доменам** (напр. «язык каждый день»).

## 6. Ранги (визуальный престиж)

Поверх числового уровня — крупные «ступени престижа» с красивыми названиями и сменой
визуала рамки/ауры. Ранг-ап — большое событие с отдельной анимацией.

| Диапазон уровней | Ранг |
|------------------|------|
| 1–9 | 🥉 Новобранец |
| 10–24 | 🥈 Искатель |
| 25–49 | 🥇 Ветеран |
| 50–99 | 💎 Мастер |
| 100+ | 🔥 Легенда |
