# 02 — Активности и квесты

← [01 — Игровые механики](./01-game-mechanics.md) · далее → [03 — Прогрессия и статы](./03-progression-and-stats.md)

Здесь — как реальные дела превращаются в игровые сущности: **активности** (атомарный
чек-ин) и **квесты** (цели поверх активностей).

---

## 1. Активности

**Активность** — единичное реальное действие, которое ты отмечаешь чек-ином. Это
базовый «кирпич» прогресса. Каждая активность задаёт: какой стат качает, базовый XP,
золото и (опционально) длительность.

### Каталог активностей (стартовый)

| Активность | Стат | База XP | Золото | Примечание |
|------------|------|---------|--------|------------|
| 🏋️ Спортзал | 💪 STR | 50 | 30 | можно указать длительность |
| 🏃 Кардио / бег | 💪 STR | 40 | 25 | |
| 🧠 Английский | 🧠 INT | 40 | 20 | урок/занятие |
| 🗣️ Испанский | 🧠 INT | 40 | 20 | урок/занятие |
| 📖 Чтение | 🧠 INT | 25 | 15 | |
| 💼 Рабочая фокус-сессия | 🎯 DIS | 35 | 20 | глубокая работа |
| ⏰ Ранний подъём | 🎯 DIS | 20 | 10 | |
| 🧘 Медитация | ❤️ VIT | 25 | 15 | |
| 😴 Здоровый сон | ❤️ VIT | 20 | 10 | 7–8 ч |
| 🥗 Здоровое питание | ❤️ VIT | 20 | 10 | |
| 🧭 Приключение / новый опыт | ✨ CHA | 70 | 50 | редкое, «эпическое» |
| 🤝 Встреча / нетворкинг | ✨ CHA | 35 | 20 | |
| 🎤 Публичное выступление | ✨ CHA | 80 | 60 | редкое |

> Каталог расширяемый. Пользователь сможет создавать **кастомные активности** (v1),
> привязав их к стату и выбрав «вес». Базовые значения и формулы — в [03](./03-progression-and-stats.md).

### Что происходит при чек-ине

```
POST /activities/checkin { activity_id, duration?, note? }
        │
        ▼
  начислить XP (×стрик-множитель, ×возможный крит)
  начислить очки стата
  начислить золото
  бросок на дроп шмота 🎁
  обновить прогресс активных квестов
  пересчитать уровень/ранг/класс
        │
        ▼
  вернуть «событие награды» → клиент проигрывает анимацию
```

Контракт чек-ина см. в [09 — API](./09-api.md).

## 2. Квесты

**Квест** — цель поверх активностей с условием выполнения и наградой сверх базовой.
Квесты дают структуру и направляют, что делать сегодня/на неделе.

### Типы квестов

| Тип | Горизонт | Пример | Сброс |
|-----|----------|--------|-------|
| **Daily** | день | «Сделай 30 мин английского» | каждый день |
| **Weekly** | неделя | «3 тренировки за неделю» | каждый понедельник |
| **Цепочка (Chain)** | длинная | «Учи испанский 30 дней подряд → титул *Conquistador*» | разовый |
| **Side / Приключение** | разово | «Попробуй что-то, чего никогда не делал» | разовый |
| **Классовый** | пока активен класс | квест под текущий класс/архетип | по классу |
| **Балансовый** | по необходимости | «Подтяни отстающий стат до LVL N» | разовый |

### Структура квеста

```yaml
id: quest_spanish_30
title: "Конкистадор"
type: chain
description: "Занимайся испанским 30 дней подряд"
condition:
  activity: spanish
  streak_days: 30
reward:
  xp: 1000
  gold: 500
  title: "Conquistador"
  item: legendary_cape_conquistador   # опциональный гарантированный дроп
icon: 🗣️
```

### Примеры квестов под домены пользователя

- **Английский / испанский (язык):**
  - Daily: «20 минут языка сегодня» (+60 XP, +25 золота)
  - Weekly: «5 языковых занятий за неделю» (+300 XP, +150 золота)
  - Chain: «Полиглот» — 14 дней любого языка подряд → титул *Полиглот*
- **Спортзал / спорт:**
  - Weekly: «3 тренировки» (+300 XP, +150 золота)
  - Chain: «Железный» — 50 тренировок суммарно → легендарный шмот *Доспех Титана*
- **Работа:**
  - Daily: «2 фокус-сессии» (+100 XP, +50 золота)
  - Chain: «Дисциплина» — 21 день подряд с фокус-сессией → титул *Несокрушимый*
- **Приключения:**
  - Side: «Открытие» — новый опыт (+150 XP, +100 золота, шанс редкого шмота)
  - Chain: «Авантюрист» — 10 приключений → класс-бонус и косметика

## 3. Ачивки (достижения)

**Ачивка** — постоянная награда-веха за пройденный рубеж. В отличие от квеста, не
сбрасывается и копится в «зале славы» профиля.

| Категория | Примеры |
|-----------|---------|
| **Старт** | Первый чек-ин · Первый level-up · Первый дроп шмота |
| **Стрики** | 🔥 7 дней · 🔥 30 дней · 🔥 100 дней |
| **Объём** | 100 тренировок · 1000 минут языка · 500 фокус-сессий |
| **Уровни** | LVL 10 · LVL 25 · LVL 50 · LVL 100 |
| **Баланс** | Все статы ≥ LVL 10 («Гармония») |
| **Домен** | Полиглот · Железный человек · Несокрушимый · Душа компании |
| **Коллекция** | Собрать сет легендарной экипировки |

Ачивки дают XP/золото/титул и иногда — эксклюзивную косметику.

## 4. Связь активность → стат → награда (сводка)

```
Активность ──даёт──▶ XP ──копит──▶ Уровень персонажа + Уровень стата
     │                                          │
     ├──даёт──▶ Золото ──тратишь──▶ Магазин (шмот)
     │
     ├──двигает──▶ Прогресс квестов ──завершение──▶ бонус XP/золото/титул/шмот
     │
     └──бросок──▶ Дроп шмота 🎁 (переменная награда)
```
